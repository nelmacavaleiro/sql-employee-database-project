# Employee Database – SQL Fundamentals

# Project Overview
This project demonstrates fundamental SQL skills through the design, population, and analysis of an employee database. It focuses on writing structured and readable SQL queries to explore employee distribution and salary patterns, reflecting real-world business reporting scenarios.

The project is implemented using SQL Server and executed via Visual Studio Code.

# Dataset Description
The database contains **50 employee records** across two departments:
- Corporate
- Private Individuals

Each record includes:
- Employee ID
- First name
- Last name
- Department
- Salary

The dataset supports salary analysis, departmental comparisons, and workforce metrics.

# Key Analytical Insights
- Total number of employees: **50**
- Total salary cost: **£219,408.24**
- Average salary: **£4,388.16**
- Maximum salary: **£6,000**
- The Corporate department represents approximately **52% of total salary expenditure**
- Corporate employees have a higher average salary than Private Individuals
- Clear salary distribution differences exist between departments

# Tools Used
- SQL Server Express
- Visual Studio Code (mssql extension)

# Techniques
- Database creation and deletion
- Table creation with primary keys
- Single-row and multi-row INSERT statements
- Filtering using WHERE, AND, BETWEEN
- Sorting using ORDER BY
- Aggregations using COUNT, SUM, AVG, MIN, MAX
- Grouped analysis using GROUP BY
- Calculated fields (bonuses and allowances)
- DISTINCT value analysis
